# aws-free-tier-monitoring

AWS Free Tier project to demonstrate cloud infrastructure + monitoring:

## What this repo covers
- EC2 provisioning (Ubuntu)
- IAM role-based access (no local AWS keys required)
- CloudWatch metrics + dashboards
- CloudWatch Logs (syslog + auth.log)
- CloudWatch alarms for alerting
- Python (boto3) scripts for validation and reporting

## Target role alignment
Built to match Systems Engineer responsibilities: automation, monitoring, Linux, AWS, documentation.
